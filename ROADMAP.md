# Roadmap

This roadmap outlines planned improvements for Persian Readability.

## v0.1.0 — Initial public release

- [x] Implement Flesch-Dayani readability score for Persian text
- [x] Add Persian normalization and tokenization
- [x] Add punctuation-aware word processing
- [x] Add optional POS-enhanced syllable counting with parsivar
- [x] Add fallback morphological syllable counting
- [x] Add command-line usage
- [x] Add tests for core readability behavior

## v0.2.0 — Packaging and usability

- [ ] Improve package structure
- [ ] Add installation instructions for PyPI-style usage
- [ ] Add more examples for education, health communication, and research
- [ ] Add sample Persian texts with expected outputs
- [ ] Add GitHub Actions for automated testing

## v0.3.0 — Documentation and validation

- [ ] Expand documentation for the Flesch-Dayani formula
- [ ] Add explanation of limitations for short texts and poetry
- [ ] Compare outputs against manually reviewed Persian text samples
- [ ] Add contribution guidelines

## Future ideas

- [ ] Build a simple web demo
- [ ] Add an API endpoint
- [ ] Add batch analysis for multiple text files
- [ ] Add CSV/JSON output
- [ ] Support readability comparison across multiple Persian documents
